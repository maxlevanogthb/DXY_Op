👓 Sistema de Gestión Integral para Óptica DXY
Una solución Fullstack diseñada para digitalizar y optimizar la operación integral de clínicas ópticas, 
garantizando la integridad de la información de los pacientes y la precisión en la gestión financiera.

🚀 Propósito del Proyecto
El sistema nace de la necesidad de centralizar la administración de una óptica, permitiendo un control estricto sobre el inventario, 
las consultas clínicas y, especialmente, el complejo flujo de ventas que implica la personalización de productos ópticos.

✨ Características Principales
Gestión de Pacientes: Control total de historias clínicas, recetas y seguimiento personalizado.
Administración de Productos: Gestión de inventarios para armazones, accesorios y suministros.
Configurador Financiero de Productos: Módulo especializado para el armado de productos compuestos (Lentes + Micas + Tratamientos + Tintes, etc.).
Automatización Contable: Cálculo automático de IVA y generación de comisiones para el personal de ventas.
Control de Ingresos: Generación de recibos, estados de cuenta y reportes de ventas en tiempo real.

🛠️ Stack TecnológicoBackend: 
Java (17+), Spring Boot, Maven.
Base de Datos: PostgreSQL.
Frontend: JavaScript (Vanilla), HTML5, CSS3.
Herramientas: Git para el control de versiones.

⚙️ Configuración e Instalación
Clonar el repositorio: git clone https://github.com/tu-usuario/optica-dxy.git.
Configurar Base de Datos: Crear una base de datos en PostgreSQL 
y actualizar las credenciales en el archivo src/main/resources/application.properties.

Compilar y Ejecutar:
mvn clean install
mvn spring-boot:run
